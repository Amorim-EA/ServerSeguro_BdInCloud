Buscar somente um unico usuario

Metodo: GET

Endpoint: localhost:3003/user

Buscar todos os usuarios

Metodo: GET

Endpoint: localhost:3003/users

Cadastrar um usuario

Metodo: POST

Endpoint: localhost:3003/user

Tipos: name: string; password: string; email: string; image: string;

Alterar um usuario

Metodo: PUT

Endpoint: localhost:3003/user/:id

Tipos: name: string; password: string; email: string; image: string;

Deletar um usuario

Metodo: DELETE

Endpoint: localhost:3003/user/:id

Criar usuario autentico com token

Metodo: POST

Endpoint: localhost:3003/user/authenticated

Tipos: email: string; password: string;
